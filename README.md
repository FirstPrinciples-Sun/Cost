## 1. โมเดลธุรกิจ: Annual Course + AI Tutor

### ภาพรวม

```
┌─────────────────────────────────────────────────────────────┐
│                  TCAS Navigator Business Model               │
│                                                              │
│  นักเรียนซื้อคอร์สเรียนรายปี (1 ครั้ง/ปีการศึกษา)          │
│  AI อาจารย์นิว ให้บริการฟรีในทุกแพ็กเกจ (ยกเว้น Free)      │
│  Learning Path แนะนำคอร์สที่เหมาะกับเด็กอัตโนมัติ          │
│  ติวเตอร์จ่ายครั้งเดียว → แพลตฟอร์มเป็นเจ้าของคอนเทนต์   │
│                                                              │
│  ช่องทางรายได้:                                             │
│  ├─ คอร์สเดี่ยว (Single Course)     ฿1,290/ปี              │
│  ├─ แพ็กวิชาวิทย์ (Science Pack)    ฿2,990/ปี              │
│  ├─ แพ็กศิลป์ (Arts Pack)          ฿1,990/ปี              │
│  └─ Full Access (ทุกคอร์ส + AI)     ฿4,990/ปี              │
└─────────────────────────────────────────────────────────────┘
```

### 7.2 NPV (Net Present Value) — Discount Rate 12%

```

ปี 0:  -$50,000 / 1.12⁰ = -$50,000

NPV = -$50,000 + (-$17,132) + $75,309 + $274,243 + $426,326 + $590,285
```

> **NPV = +$1,299,031** → ลงทุนคุ้มค่ามาก ✅

### 7.4 Payback Period

```

ปี 2: +$25,263     ← ครบเริ่มกำไร!

```

### 7.5 ROI (Return on Investment)

```
Initial Investment (Seed) = $50,000

5-Year ROI = ($2,070,770 - $50,000) / $50,000 = 40.41 = 4,041%
Annualized ROI = (1 + 40.41)^(1/5) - 1 = 1.4041^0.2 - 1 = 36.6% ต่อปี
```

> **5-Year ROI = 4,041%** | **Annualized = 36.6%** ✅
