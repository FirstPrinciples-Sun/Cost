## 🎯 หลักการเรียงลำดับ Pitch Deck ที่ดี

### The Story Arc (โครงเรื่อง)

```
1. เปิดด้วย "ทำไม" (Why)     → ปัญหา + โอกาส
2. ตามด้วย "อะไร" (What)     → ทางออก + สินค้า
3. ต่อด้วย "อย่างไร" (How)   → ธุรกิจ + การเติบโต
4. ปิดด้วย "ใคร" (Who)       → ทีม + การลงทุน
```

### กฎ 3 นาทีแรก

ผู้ลงทุนส่วนใหญ่ตัดสินใจ **ภายใน 3 นาทีแรก** ว่าจะสนใจต่อหรือไม่

ดังนั้น 3 สไลด์แรกต้อง:
- ✅ ปัญหาชัดเจน วัดผลได้
- ✅ ทางออกเข้าใจง่าย dalam 10 วินาที
- ✅ ตลาดใหญ่พอที่จะน่าสนใจ
